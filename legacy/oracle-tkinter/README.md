# Legacy Oracle/Tkinter application

This directory preserves the original academic Race Data Management System:

- `main.py` — Tkinter client, embedded Flask routes and Oracle queries
- `racedata.sql` — Oracle schema
- `sample_data.sql` — original 2021 sample dataset

It requires Oracle 21c XE, `cx_Oracle`, and the credentials configured inside `main.py`. It is retained for reference only and is not part of the React/FastAPI runtime.
